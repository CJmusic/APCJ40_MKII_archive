# APCJ40_MKII

A custom remote script for the APC40 MKII for Ableton Live 11. 

So far the only changes are: 

- nudge + copies a clip 
- nudge - deletes a clip 
- shift + nudge now nudges 

Changes to come: 

- Step Sequencer
